My Projects

Simple Portfolio

The portfolio was built using HTML, CSS, Bootstrap, and JavaScript, allowing for smooth and easy navigation through images. When an image is clicked, it is displayed along with its position number in the gallery. Users can navigate through the rest of the images using arrow buttons or close the image view with the close icon.

Shop Website

I designed an application similar to Amazon using HTML, CSS, Bootstrap, and JavaScript. The app displays products, allowing users to select the desired product, choose the color, size, and quantity, and add it to the cart. All items placed in the cart are displayed on the side along with the total price, and users can also remove items from the cart.

Food Website

A food website built using HTML, CSS, JavaScript, and an API displays meals based on the user's selection of meal type, drinks, salads, or desserts. The meal data is fetched from an API. When the user clicks on "Details," more information about the selected meal is shown.

School Website

The school website, built with HTML, CSS, Bootstrap, React JS, Redux Toolkit, and backend APIs, offers materials, units, and lessons for all grades. Selecting a class displays relevant content like lessons, worksheets, and videos. The site includes a login page: admins can manage sub-admins and content permissions, while sub-admins handle content within their granted permissions. Additionally, it features a dark mode for a better user experience.
